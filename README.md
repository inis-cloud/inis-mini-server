## 安装依赖

```bash
$ yarn 或 yarn install
$ npm i 或 npm install
```

## 运行

```bash
# development
$ yarn start
$ npm run start

# watch mode
$ yarn start:dev
$ npm run start:dev

# production mode
$ yarn start:prod
$ npm run start:prod
```
